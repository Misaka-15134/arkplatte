Arknights palette tutorial

明日方舟科研配色教程

库简介

arkplatte 为明日方舟角色主题科研绘图提供 Python 库和 R 包，适合连续数值、正负差异、分组注释、热图、火山图、UMAP 和 plot1cell 等常见场景。

配色采用固定校准表，覆盖 417 名干员；头像近景优先用于角色本体取色，立绘作为备用来源。函数读取固化后的 palettes.csv，保证同一角色在不同图中保持稳定视觉语义；其中 131 名六星干员保留人工校准表。

色阶生成结合 Lab 感知色彩空间插值、亮度单调约束、双向色阶高亮中点、两端亮度对称和分类色最大距离选择。单细胞数据可用大类绑定角色主题，小类在同一主题内自动挑选距离更合适的颜色，从而兼顾层级关系和类别区分度。

函数及使用方法

arkplatte(name, n)
角色核心色。
arkplatte_seq(name, n)
单向连续色。
arkplatte_div(name, n)
双向连续色。
arkplatte_cat(n)
跨角色分类色。
arkplatte_cell(celltypes)
细胞类型分类色，适合单细胞分析实验。
arkplatte_sub(map, operator)
大类绑定角色，小类按数量自适应选色。
curated_palettes.csv
六星人工校准表。

核心色与色阶入口

核心 / 单向 / 双向
Pythonakp.arkplatte(name, n=None, kind="core")
Rarkplatte(name, n = NULL, type = "core")
参数name：中文名、name_key 或干员编号。
n：颜色数量。
kind/type："core"、"seq"、"div"。
调节内容读取角色核心色；设置 n 时插值到指定数量；可直接切换核心色、单向连续色或双向连续色。
返回值颜色向量或颜色列表。

单向连续色

0 到 1 / 表达量 / 丰度
Pythonakp.arkplatte_seq(name, n=256)
Rarkplatte_seq(name, n = 256)
参数name：角色。
n：连续色数量，教程示例常用 5 到 10。
调节内容生成亮度单调的单向连续色；Lab 插值让亮度从浅到深平滑变化。
返回值亮度单调色阶。

双向连续色

-1 到 1 / 差异 / 相关系数
Pythonakp.arkplatte_div(name, n=257)
Rarkplatte_div(name, n = 257)
参数name：角色。
n：双向色数量，教程示例常用 7 或 9。
调节内容生成中点高亮、两端亮度对称的双向连续色，适合有正负方向的数值。
返回值双向色阶。

跨角色分类色

分组 / 注释 / 图例
Pythonakp.arkplatte_cat(n, seed=None, large_n="warn", optimize=True)
Rarkplatte_cat(n, seed = NULL, large_n = "warn", optimize = TRUE)
参数n：类别数量。
seed：起始偏移。
large_n：超过 30 类时的策略。
optimize：是否按 Lab 距离优化。
调节内容生成跨角色分类色；默认优先挑距离更远的颜色；超过 30 类时提醒切换到分组、分面或上层注释。
返回值分类颜色向量。

细胞类型分类色

单细胞分析实验
Pythonakp.arkplatte_cell(celltypes, seed=1, large_n="warn")
Rarkplatte_cell(celltypes, seed = 1, large_n = "warn")
参数celltypes：细胞类型列表。
seed：分类色偏移。
large_n：大类别数策略。
调节内容给细胞类型名称直接分配颜色，适合 UMAP、t-SNE 和细胞注释图。
返回值名称到颜色的映射。

大类与小类分层配色

粗分类 / 细分类
Pythonakp.arkplatte_sub(subtype_map, group_operator, candidates_per_group=96)
Rarkplatte_sub(subtype_map, group_operator, candidates_per_group = 96)
参数subtype_map：大类到小类列表。
group_operator：大类到角色。
candidates_per_group：每个大类候选色数量。
调节内容为每个大类绑定一个角色主题；按小类数量扩展候选池，再按色彩距离挑选更分散的颜色。
返回值小类名称到颜色的映射。

arkplatte_info(name)

name：角色查询键。
调节内容查看可用干员名称，或查询角色编号、中文名、职业、分支和数据来源。

主题色提取

多角色主色

Cell-type mapping
